# Travel App

## App Description
Travel App is a simple Android application created using Kotlin and Jetpack Compose.  
The app allows users to add travel packing items, view the packing list, and display items with quantities greater than or equal to 2.

## Features
- Add travel items
- Enter category
- Enter quantity
- Add comments
- Display packing list
- Filter items with quantity greater than or equal to 2
- Navigate between two screens
- Exit application

## Screens

### Screen 1
- Enter item information
- Add item button
- Navigate to second screen
- Exit button

### Screen 2
- Display packing list
- Show items with quantity >= 2
- Back button

## Technologies Used
- Kotlin
- Android Studio

## How To Run The App
1. Open the project in Android Studio.
2. Build and run the application.
3. Use an emulator or Android device.
4. Enter travel items and test the app.
